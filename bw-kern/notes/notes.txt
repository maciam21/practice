Hot Keys 
1. copy = cntrl + C
2. cut = cntrl + X
3. paste = Cntrl + V
4. undo = cntrl + Z
5. select all = cntrl + A
6. Save = Cntrl + S
Bonus:
    redo = cntrl + Y (sometimes need shift)
    find = Cntrl + F
    Delete = Cntrl + D
windows screenshot = windows logo key + shift + S
